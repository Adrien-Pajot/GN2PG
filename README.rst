.. contents:: Topics

.. warning::
    Actually not usable, in active development.


Project Setup
=============

Gn2Gn Client can be installed by running ``pip``. It requires Python 3.7.4+ to run.

.. code-block:: bash

    python3 -m pip install https://github.com/lpoaura/gn2gn_client/archive/dev.zip
